## Run Locally

Clone the project

```bash
  git clone https://github.com/AngelPASTORROJAS/solair.git
```

Go to the project directory

```bash
  cd solair
```

Install dependencies

```bash
  npm install
```

Build the project

```bash
  npm run build
```


Start the server

```bash
  npm run start
```
